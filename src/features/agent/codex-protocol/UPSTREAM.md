# Codex App Server protocol types

This directory contains TypeScript protocol definitions generated by the installed official Codex CLI.

- Generator: `codex-cli 0.144.1` (output verified byte-for-byte against `0.144.5`)
- Command: `codex app-server generate-ts --out src/features/agent/codex-protocol/generated`
- Upstream: https://github.com/openai/codex/tree/main/codex-rs/app-server-protocol
- License: Apache-2.0

Files under `generated/` are generated code and must not be edited by hand. Aryn intentionally uses the stable protocol surface and initializes App Server with `experimentalApi: false`; do not add `--experimental` unless the runtime capability and all added requests are implemented together. Regenerate and review the resulting protocol diff when upgrading the supported Codex CLI version.

Run `npm run verify:codex-protocol` to verify that the complete generated tree still matches the pinned aggregate hash.
